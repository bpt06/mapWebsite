Arduino Microcontroller

This is an Arduino Uno microcontroller. A microcontroller connects to a computer to download a program and then can interact with electronic components through the digital and analog I/O pins. There are many forms of microcontrollers, but the Arduino Uno is one of the most common and easiest-to-use examples. Microcontrollers are in almost every electronic device that you use on a day-to-day basis.

IC/Integrated Circuit

An IC, or integrated circuit, is a processor chip with inputs and outputs and an internal circuit that does some form of calculation depending on the type of chip. In this case, it is a 74HC595 IC, which takes a single serial input and breaks it into 8 parallel outputs. One of the main reasons that IC chips are so useful is that you don't need to know what is going on inside of them, you only have to know how to use them.

Resistors

A resistor is a component used in electronics that adds resistance to a circuit. This is important when multiple components in a circuit have different power limits. In this example, all the leads coming from the IC and going to the seven-segment display have 1kohm resisters, so that the voltage going into the display isn't more than it can handle, possibly harming components.

Seven Segment Display

A seven-segment display is a display primarily used to show numbers. It has seven individual segments that can be lit up, and with them, you can display any single-digit number. These displays are commonly found in digital clocks. A single-digit seven-segment display such as is used here has two communication pins, that connect to the microcontroller to tell the display when to light up, as well as one pin for each of the seven segments. To make the display show a certain number you have to apply a charge to the pins for the segments you want on, and then send a signal through the communication pins to tell the display to light up the given segments.

How is a Circuit Diagram a Map?

There are many definitions for what a map is. The definition that I use is that a map is a graphic statement that locates facts. This is a very broad definition, which allows for more creativity when making new maps. The genre that this map falls into is a Circuit Diagram. Circuit diagrams are maps showing electronic components and the connections between them. These maps are very important for understanding a circuit, and for someone else to be able to recreate it. While the components are roughly in the correct place, the distances between them don't matter, it's just the connections that are important. Circuit diagrams can be made in many different ways, for many different purposes. The example above is a map that I made that is designed to give a realistic representation of what all of the components look like when the circuit is assembled. Other diagrams are much more scientific and are just boxes that are labeled with what they represent, instead of benign pictures of what they are such as in Example 1.
